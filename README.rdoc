== README

This Application is a sample of a card game I created. To see a running version go to:

https://marko-g-card-siege.herokuapp.com/

Rails mechanics used:

* Concerns
* AJAX communication from js to controller and back
* getJSON to get values from controller for pages
* Partials
* view helpers
* HTML Bootstrap
* JQuery
* running js functions from erb files
* validates
* has_many, belongs_to relationships
